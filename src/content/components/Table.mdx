## Sobre el componente

Este componente se utilizar para mostrar información tabular utilizando filas y columnas, permitiendo al usuario visualizar la información de manera rapida y organizada.

## Uso básico

Para utilizar el componente `Table` es necesario hacer uso de `TableHeader`, `TableBody`, `TableRow`, `TableCell` y `TableColumn` para estructurar la información.

<Preview>
  <CustomTable client:load />
</Preview>

<CodeBlock client:load>

```jsx
<Table>
  <TableHeader>
    <TableColumn>id</TableColumn>
    <TableColumn>name</TableColumn>
    <TableColumn>age</TableColumn>
  </TableHeader>
  <TableBody>
    <TableRow id='1'>
      <TableCell>1</TableCell>
      <TableCell>Jhon Doe</TableCell>
      <TableCell>28</TableCell>
```

## Tabla vacía

Para mostrar una tabla vacía es necesario utilizar el atributo `isEmpty` en el componente `TableBody` y no debe contener ningún elemento dentro de él. Además, es posible utilizar el atributo `emptyMessage` para mostrar un mensaje personalizado en caso de que la tabla esté vacía.

<Preview>
  <CustomTable empty emptyMessage='No existen elementos.' />
</Preview>

<CodeBlock client:load>

```jsx
<Table>
  <TableHeader>
    <TableColumn>id</TableColumn>
    <TableColumn>name</TableColumn>
    <TableColumn>age</TableColumn>
  </TableHeader>
  <TableBody isEmpty emptyMessage='No existen elementos.'>
    {[]}
  </TableBody>
</Table>
```

</CodeBlock>

## Tabla con datos

Para mostrar una tabla con datos es necesario utilizar el atributo `isLoading` en el componente `TableBody` . Además, es posible utilizar el atributo `loadingContent` para mostrar un mensaje personalizado en caso de que la tabla esté cargando los datos.
