## Sobre el componente

Este componente se utiliza para permitir al usuario seleccionar una fecha.

## Uso básico

Para utilizar el componente, simplemente debes definir el atributo `value`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <DateInput client:load value='2023-01-01' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<DateInput value='2023-01-01' />
```

</CodeBlock>
