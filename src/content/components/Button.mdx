## Sobre el componente

Este componente se utiliza para llevar a cabo una acción en un sitio web con un simple clic.

## Uso básico

Para utilizar el componente, simplemente debes definir el atributo `text`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load text='Default' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Button text='Default' />
```

</CodeBlock>

El código para `Button` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: ButtonReact },
    { label: 'TypeScript', code: ButtonTs }
  ]}
/>

## Variantes

Para cambiar el estilo del componente, se puede utilizar el atributo `variant` en el componente `Button`. Este atributo acepta cuatro valores: `default`, `bordered`, `light` y `complete`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load text='Default' variant='default' />
    <Button client:load text='Bordered' variant='bordered' />
    <Button client:load text='Light' variant='light' />
    <Button client:load text='Complete' variant='complete' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Button text='Default' variant='default' />
<Button text='Bordered' variant='bordered' />
<Button text='Light' variant='light' />
<Button text='Complete' variant='complete' />
```

</CodeBlock>

## Icono

Para agregar un icono al componente, simplemente debes agregar el atributo `icon={true}` en el componente `Button`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load text='Icon' color='danger' icon={true} />
  </div>
</Preview>

<CodeBlock client:load>

</CodeBlock>
