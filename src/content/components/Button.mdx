## Sobre el componente

Este componente se utiliza para llevar a cabo una acción en un sitio web con un simple clic.

## Uso básico

Para utilizar el componente, simplemente debes definir el contenido del componente.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load>Default</Button>
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Button>Default</Button>
```

</CodeBlock>

El código para `Button` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: ButtonReact },
    { label: 'TypeScript', code: ButtonTs }
  ]}
/>

## Variantes

Para cambiar el estilo del componente, se puede utilizar el atributo `variant` en el componente `Button`. Este atributo acepta cuatro valores: `default`, `bordered`, `light` y `complete`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load variant='default'>
      Default
    </Button>
    <Button client:load variant='bordered'>
      Bordered
    </Button>
    <Button client:load variant='light'>
      Light
    </Button>
    <Button client:load variant='complete'>
      Complete
    </Button>
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Button variant='default'>Default</Button>
<Button variant='bordered'>Bordered</Button>
<Button variant='light'>Light</Button>
<Button variant='complete'>Complete</Button>
```

</CodeBlock>

## Botón de carga

Para mostrar un botón de carga, simplemente debes agregar el atributo `isLoading={true}` en el componente `Button`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Button client:load color='warning' isLoading={true}>
      Loading
    </Button>
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Button color='warning' isLoading={true}>
  Loading
</Button>
```

</CodeBlock>

      Disabled
    </Button>
  </div>
