## Sobre el componente

Este componente se utiliza para permitir al usuario seleccionar múltiples elementos de una lista de elementos individuales o seleccionar un elemento individual.

## Uso básico

Para utilizar el componente, simplemente debes definir los atributos `id` y `label`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Checkbox client:load id='checkbox-1' label='Checkbox' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Checkbox id='checkbox-1' label='Checkbox' />
```

</CodeBlock>

El código para `Checkbox` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: CheckboxReact },
    { label: 'TypeScript', code: CheckboxTs },
    { label: 'Astro', code: CheckboxAstro }
  ]}
/>

## Variantes

Para cambiar el estilo del componente, se puede utilizar el atributo `variant` en el componente `Checkbox`. Este atributo tiene 3 valores posibles: `default`, `bordered` y `light`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Checkbox client:load id='checkbox-2' label='Checkbox' variant='default' />
    <Checkbox client:load id='checkbox-3' label='Checkbox' variant='bordered' />
    <Checkbox client:load id='checkbox-4' label='Checkbox' variant='light' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Checkbox id='checkbox-2' label='Checkbox' variant='default' />
<Checkbox id='checkbox-3' label='Checkbox' variant='bordered' />
<Checkbox id='checkbox-4' label='Checkbox' variant='light' />
```

</CodeBlock>

## Seleccionado

Para indicar que el componente está seleccionado, se puede utilizar el atributo `checked = {true}` en el componente `Checkbox`. Este atributo tiene un valor booleano que determina si el componente está seleccionado o no.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>

  </div>
</Preview>
