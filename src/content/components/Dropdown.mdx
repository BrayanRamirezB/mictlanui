## Sobre el componente

Este componente se utiliza para mostrar un menú desplegable con múltiples opciones al usuario.

## Uso básico

Para utilizar el componente, debemos hacer uso de los componentes `Dropdown`, `DropdownTrigger`, `DropdownMenu`, `DropdownSection` y `DropdownItem`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <CustomDropdown client:load />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Dropdown>
  <DropdownTrigger>Open Dropdown</DropdownTrigger>
  <DropdownMenu>
    <DropdownSection heading='Section 1'>
      <DropdownItem title='Titulo 1' description='decripcion 1' />
      <DropdownItem title='Titulo 2' description='decripcion 2' />
    </DropdownSection>
  </DropdownMenu>
</Dropdown>
```

</CodeBlock>

Y el código para el componente `Dropdown` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: DropdownReact },
    { label: 'TypeScript', code: DropdownTs }
  ]}
/>

El código para el componente `DropdownTrigger` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: DropdownTriggerReact },
    { label: 'TypeScript', code: DropdownTriggerTs }
  ]}
/>

El código para el componente `DropdownMenu` es el siguiente:

  client:load
