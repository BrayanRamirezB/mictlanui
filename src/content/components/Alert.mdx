## Sobre el componente

Este componente se utiliza para mostrar información relevante a los usuarios como mensajes de error, advertencias o éxito. También puede utilizarse para mostrar información adicional o para darle a los usuarios una respuesta a una pregunta.

## Uso básico

Para utilizar el componente, simplemente debes definir los atributos `title` y `content`.

<Preview>
  <div class='w-full m-auto'>
    <Alert client:load title='Alert' content='This is a default alert.' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Alert title='Alert' content='This is a default alert.' />
```

</CodeBlock>

El código para `Alert` es el siguiente:

<CodeTabs

    { label: 'TypeScript', code: AlertTs }

/>

## Tipos de alertas

Para cambiar el estilo del componente, se puede utilizar el atributo `type` en el componente `Alert`. Este atributo acepta seis valores: `default`, `primary`, `secondary`, `success`, `warning` y `danger`.

El estilo por defecto del componente es el `default`.

## Variantes

Para cambiar el estilo del componente, se puede utilizar el atributo `styleVariant` en el componente `Alert`. Este atributo acepta cuatro valores: `default`, `light`, `bordered` y `complete`.

La variante por defecto del componente es `default`.

<Preview>

    <Alert
      client:load
      title='Default style variant'
      styleVariant='default'
      type='primary'
    />
    <Alert
      client:load
      title='Light style variant'
      styleVariant='light'
      type='primary'
    />
    <Alert
      client:load
      title='Bordered style variant'
      styleVariant='bordered'
      type='primary'
    />
    <Alert
      client:load
      title='Complete style variant'
      styleVariant='complete'
      type='primary'
    />
  </div>
</Preview>

## Icono

Para ocultar el icono del componente, simplemente debes agregar el atributo `icon={false}` en el componente `Alert`.

<Preview>
  <div class='w-full m-auto'>
    <Alert
      client:load
      title='Alert'
      content='This is a default alert.'
      icon={false}
    />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Alert title='Alert' content='This is a default alert.' icon={false} />
```

</CodeBlock>

## Botón de cierre

Para ocultar el componente, simplemente debes agregar el atributo `dismissible={true}` en el componente `Alert`.

<Preview>
  <div class='w-full m-auto'>
    <Alert
      client:load
      title='Alert'
      content='This is a default alert.'
      type='secondary'
      dismissible={true}
    />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Alert
  title='Alert'
  content='This is a default alert.'
  type='secondary'
  dismissible={true}
/>
```

</CodeBlock>
