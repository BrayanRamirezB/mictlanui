## Sobre el componente

Este componente se utiliza para mostrar información relevante a los usuarios como mensajes de error, advertencias o éxito. También puede utilizarse para mostrar información adicional o para darle a los usuarios una respuesta a una pregunta.

## Uso básico

Para utilizar el componente, simplemente debes definir los atributos `title` y `content`.

<Preview>
  <div class='w-full m-auto'>

  </div>
</Preview>

## Tipos de alertas

Para cambiar el estilo del componente, se puede utilizar el atributo `type` en el componente `Alert`. Este atributo acepta seis valores: `default`, `primary`, `secondary`, `success`, `warning` y `danger`.

El estilo por defecto del componente es el `default`.

## Variantes
