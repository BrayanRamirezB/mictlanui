## Sobre el componente

Este componente se puede utilizar para permitir a los usuarios ingresar sus datos en función de varios tipos de entrada, como texto, correo, número, contraseña, etc.

## Uso básico

Para utilizar este componente, agrega el componente `Input` con los atributos `placeholder`, `value` y `onValueChange` junto con un estado para manejar el valor del campo de texto.

<Preview>
  <CustomInput placeholder='Ingresa un texto' client:load />
</Preview>

<CodeBlock client:load>

```jsx
const [value, setValue] = useState('')

return (
  <Input
    placeholder='Ingresa un texto'
    value={value}
    onValueChange={setValue}
  />
)
```

</CodeBlock>

El código para el componente `Input` es el siguiente:

## Longitud mínima y máxima

Para establecer una longitud mínima y máxima para el campo, puedes utilizar el atributo `minLength` y `maxLength` respectivamente.

<Preview>
  <CustomInput
    label='Longitud mínima y máxima'
    placeholder='Ingresa un texto'
    isInvalid
    isMinLength
    errorMessage={'El campo debe tener al menos 5 caracteres'}
    minLength={5}
    maxLength={10}
    client:load
  />
</Preview>

<CodeBlock client:load>

```jsx
const [value, setValue] = useState('')

return (
  <Input
    label='Longitud mínima y máxima'
    placeholder='Ingresa un texto'
    isInvalid={value.length < 5}
    errorMessage={'El campo debe tener al menos 5 caracteres'}
    minLength={5}
    maxLength={10}
    value={value}
    onValueChange={setValue}
  />
)
```

</CodeBlock>
