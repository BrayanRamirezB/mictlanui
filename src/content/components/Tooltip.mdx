## Sobre el componente

Este componente permite al usuario mostrar información contextual o algun tipo de ayuda para comprender mejor un elemento.

## Uso básico

Para utilizar el componente, simplemente debemos definir el atributo `content` del componente `Tooltip`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Tooltip client:load content='Este es un tooltip'>
      <Button text='Botón con tooltip' />
    </Tooltip>
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Tooltip content='Este es un tooltip'>
  <Button text='Botón con tooltip' />
</Tooltip>
```

</CodeBlock>

El código para el componente `Tooltip` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: TooltipReact },
    { label: 'TypeScript', code: TooltipTs }
  ]}
/>

## Posicionamiento

Para cambiar el posicionamiento del componente, se puede utilizar el atributo `placement` en el componente `Tooltip`. Este atributo los siguientes valores: `top`, `bottom`, `left` y `right`.

    </Tooltip>

    <Tooltip client:load content='Este es un tooltip' placement='top'>

## Delay

Para cambiar el tiempo de espera antes de mostrar el componente, se puede utilizar el atributo `delay`. Este atributo acepta un valor en milisegundos.

      <Button text='Tooltip with delay 1000ms' />
    </Tooltip>

      <Button text='Tooltip with delay 2000ms' />
    </Tooltip>

## Deshabilitado

Para deshabilitar el componente, se puede utilizar el atributo `isDisabled = {true}` en el componente `Tooltip`.

<Preview>

    <Tooltip client:load content='Este es un tooltip' isDisabled={true}>
      <Button text='Disabled tooltip' />
    </Tooltip>
  </div>
</Preview>

<CodeBlock client:load>


</CodeBlock>
