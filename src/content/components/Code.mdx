## Sobre el componente

Este componente se utiliza para mostrar código en linea.

## Uso básico

Para utilizar el componente, simplemente debes definir los atributos `codeString` y `language`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <Code client:load codeString='<div>Hola Mundo</div>' language='html' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Code codeString='<div>Hola Mundo</div>' language='html' />
```

</CodeBlock>

El código para `Code` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: CodeReact },
    { label: 'TypeScript', code: CodeTs }
  ]}
/>

## Variantes

Para cambiar el estilo del componente, se puede utilizar el atributo `variant` en el componente `Code`. Este atributo tiene los siguientes valores: `default`, `bordered` y `light`.

        codeString='<div>Hola Mundo</div>'
        language='html'

      />
    </div>
    <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
      <Code
        client:load

      <Code
        client:load
