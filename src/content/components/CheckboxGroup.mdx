## Sobre el componente

Este componente se utiliza para permitir al usuario seleccionar uno o más elementos de una lista de opciones.

## Uso básico

Para utilizar el componente, simplemente debes definir los atributos `title` y `checkboxes`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <CheckboxGroup
      client:load
      title='CheckboxGroup'
      checkboxes={[
        { id: 'checkbox-1', label: 'Checkbox 1' },
        { id: 'checkbox-2', label: 'Checkbox 2' },
        { id: 'checkbox-3', label: 'Checkbox 3' }
      ]}
    />
  </div>
</Preview>

<CodeBlock client:load>
