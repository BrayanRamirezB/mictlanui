## Sobre el componente

Este componente se utiliza para agrupar multiples botones en un solo componente.

## Uso b치sico

Para utilizar el componente, simplemente debes definir el atributo `buttons` en el componente `ButtonGroup`. Este atributo acepta un array de objetos con la propiedad `label` y `icon`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <ButtonGroup
      client:load
      buttons={[
        {
          label: 'Profile'
        },
        {
          label: 'Settings'
        },
        {
          label: 'Downloads'
        },
        {
          label: 'Logout'
        }
      ]}
    />
  </div>
</Preview>
