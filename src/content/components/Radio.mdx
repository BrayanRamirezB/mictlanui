## Sobre el componente

El componente `Radio` es una herramienta de selección de opciones que permite al usuario seleccionar una opción de una lista de opciones.

## Uso básico

Para utilizar el componente `RadioGroup`, es necesario utilizar el componente `Radio` con el atributo `value` dentro del componente, además de un estado para manejar la selección del usuario.

<Preview>
  <CustomRadio client:load />
</Preview>

<CodeBlock client:load>

```jsx
const [selectedValue, setSelectedValue] = useState(null)

const handleChange = (value) => {
  setSelectedValue(value)
}

return (
  <div className='p-4'>
    <RadioGroup selectedValue={selectedValue} onChange={handleChange}>
      <Radio value='option1'>Option 1</Radio>
      <Radio value='option2'>Option 2</Radio>
      <Radio value='option3'>Option 3</Radio>
    </RadioGroup>
  </div>
)
```

</CodeBlock>

## Error de validación

Para agregar un mensaje de error al componente, simplemente debes agregar el atributo `errorMessage` en el componente `RadioGroup`. Para indicar que el componente es inválido, se puede utilizar el atributo `isInvalid` en el componente `RadioGroup`. Este atributo acepta un booleano que determina si el componente es inválido o no.

<Preview>
  <CustomRadio
    client:load
    errorMessage='Debes seleccionar la opción 2'
    IsInvalid={true}
  />
</Preview>

<CodeBlock client:load>

```jsx
const [selectedValue, setSelectedValue] = useState(null)

const handleChange = (value) => {
  setSelectedValue(value)
}

const validOptions = ['option2']

const isInvalid = !validOptions.includes(selectedValue)

return (
  <div className='p-4'>
    <RadioGroup
      selectedValue={selectedValue}
      onChange={handleChange}
      errorMessage='Debes seleccionar la opción 2'
      isInvalid={isInvalid}
    >
      <Radio value='option1'>Option 1</Radio>
      <Radio value='option2'>Option 2</Radio>
      <Radio value='option3'>Option 3</Radio>
    </RadioGroup>
  </div>
)
```

</CodeBlock>
