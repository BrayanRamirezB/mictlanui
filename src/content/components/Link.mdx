## Sobre el componente

Este componente se puede utilizar para establecer hipervínculos en el sitio web o para redireccionar a otra página web.

## Uso básico

Para establecer un hipervínculo, simplemente agrega el atributo `href` con el valor que desee.

<Preview>
  <Link href='#'>Enlace</Link>
</Preview>

<CodeBlock client:load>

```jsx
<Link href='#'>Enlace</Link>
```

</CodeBlock>

El código para el componente `Link` es el siguiente:

Este componente posee algunas variantes que puedes utilizar según tus necesidades. Para esto, puedes utilizar el atributo `variant` y cambiar su valor entre `default`, `bordered` o `light`.
