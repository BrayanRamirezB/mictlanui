## Sobre el componente

Este componente se utiliza para organizar contenido en multiples secciones permitiendo al usuario navegar entre dichas secciones.

## Uso básico

Para utilizar el componente, simplemente debemos utilizar el componente `Tabs` y dentro de él, el componente `Tab`.

<Preview>
  <div class='flex items-center justify-center gap-x-2 w-full m-auto'>
    <CustomTab client:load />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Tabs>
  <Tab label='Tab 1'>
    <div>Contenido de la pestaña 1</div>
  </Tab>
  <Tab label='Tab 2'>
    <div>Contenido de la pestaña 2</div>
  </Tab>
  <Tab label='Tab 3'>
    <div>Contenido de la pestaña 3</div>
  </Tab>
</Tabs>
```

</CodeBlock>

El código para el componente `Tab` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: TabsReact },
    { label: 'TypeScript', code: TabsTs }
  ]}
/>

El código para el componente `Tabs` es el siguiente:

<CodeTabs
  client:load
  tabs={[
    { label: 'React', code: TabsReact },
    { label: 'TypeScript', code: TabsTs }
  ]}
/>

<Preview>
  <div class='flex flex-col items-center justify-center gap-y-2 w-full m-auto'>
    <CustomTab client:load disabled={true} />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Tabs disabled={true}>
  <Tab label='Tab 1'>
    <div>Contenido de la pestaña 1</div>
  </Tab>
  <Tab label='Tab 2'>
    <div>Contenido de la pestaña 2</div>
  </Tab>
  <Tab label='Tab 3'>
    <div>Contenido de la pestaña 3</div>
  </Tab>
</Tabs>
```

</CodeBlock>

## Deshabilitado de una pestaña

Para deshabilitar un pestaña en particular, simplemente debes agregar el atributo `disabled={true}` en el componente `Tab`.

<Preview>
  <div class='flex flex-col items-center justify-center gap-y-2 w-full m-auto'>
    <CustomTab client:load isDisabled={true} />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Tabs>
  <Tab label='Tab 1'>
    <div>Contenido de la pestaña 1</div>
  </Tab>
  <Tab label='Tab 2' disabled={true}>
    <div>Contenido de la pestaña 2 (deshabilitada)</div>
  </Tab>
  <Tab label='Tab 3'>
    <div>Contenido de la pestaña 3</div>
  </Tab>
</Tabs>
```

</CodeBlock>

## Enlace

Para agregar un enlace a una pestaña, simplemente debes agregar el atributo `href` en el componente `Tab`.

<Preview>
  <div class='flex flex-col items-center justify-center gap-y-2 w-full m-auto'>
    <CustomTab client:load href='#' />
  </div>
</Preview>

<CodeBlock client:load>

```jsx
<Tabs>
  <Tab label='Tab 1'>
    <div>Contenido de la pestaña 1</div>
  </Tab>
  <Tab label='Tab 2'>
    <div>Contenido de la pestaña 2 (enlace)</div>
  </Tab>
  <Tab label='Tab 3' href='#'>
    <div>Contenido de la pestaña 3</div>
  </Tab>
</Tabs>
```

</CodeBlock>
